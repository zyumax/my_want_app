# アプリ名：「Wantest」
自分の欲しいものを管理して支出予測するアプリです。
[Wantest](https://my-want.life/)

# 実装機能
* 新規ユーザー登録機能、ログイン機能
* 商品登録、編集、削除機能
* 登録した商品の合計金額表示機能
